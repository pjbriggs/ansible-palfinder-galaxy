Centaurus Galaxy Server News

[13/03/2024] Centaurus updated to Galaxy version 21.05 and moved to new host VM

The Centaurus Galaxy service has been upgraded to Galaxy version 21.05 and moved to a new host virtual machine (VM) running Ubuntu Linux.

Full details of changes in release 21.05 of Galaxy can be found in the official announcement from the Galaxy Project for this version:

  • Galaxy 21.05

The VM update was required as the previous operating system was no longer supported and was not up-to-date with security patches.

If you encounter any problems with the service following the updates then please contact me via peter.briggs@manchester.ac.uk. Where appropriate it is also helpful if you can share the history containing the problem with me within Galaxy.

[02/12/2014] Centaurus Galaxy now running

The initial version of the Centaurus Galaxy server is now up and running.
